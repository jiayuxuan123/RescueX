RescueX 通过监控启动失败次数和开机超时，自动禁用问题模块以救砖。兼容 Magisk / KernelSU / APatch。基于 uptime 单调时钟计算启动耗时，不受 RTC 同步影响。v3.2.4 继续整合快照、稳定基线、决策报告、高风险脚本拦截与 GitHub 代理更新。
